# 豆瓣图书api 返回403
## vue工程的index.html文件中加上
### 可以去除防盗链
```html
<meta name="referrer" content="never" />
```
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width,initial-scale=1.0">
    <meta name="referrer" content="never" />
    <title>bookvue</title>
  </head>
  <body>
    <div id="app"></div>
    <!-- built files will be auto injected -->
  </body>
</html>
```
